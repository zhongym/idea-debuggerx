# DebuggerX

<small>支持IDE版本: `2023.*`</small>

-----
基于[debugger-enhancer](https://github.com/lppedd/idea-debugger-enhancer)插件进行修改。
- 支持最新的idea
- 支持idea原来的debug Condition
- 添加"自动执行表达式"功能

这个插件提供了额外的操作来调试断点。 
目前，它支持:

- 强制返回任意值
- 强制抛出异常
- 自动执行表达式
